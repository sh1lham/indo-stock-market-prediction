# 🇮🇩 IDX Multi-Model Stock Forecasting Dashboard

Web-based stock price forecasting application for Indonesian equities 
(BBCA, BBRI, BUMI) on Bursa Efek Indonesia.

## Models Implemented
- LSTM (best performer — 2.84% MAPE, 76.2% directional accuracy)
- ARIMA
- Transformer
- Reinforcement Learning Agent
- Moving Averages (MA50 baseline)

## Features
- Configurable 20/30/60-day prediction windows
- Interactive actual vs. predicted price chart (120 trading days)
- Model comparison metrics table
- Buy/sell signal generation
- Live simulation with IDX reference prices

## Tech Stack
HTML, CSS, JavaScript, Python
